# Project 04: Inventory and Operations Insights

## Goal
Analyze product sales patterns to understand inventory movement, demand consistency, and operational risk across products and categories.

## Dataset
BigQuery public dataset: bigquery-public-data.thelook_ecommerce

## Tables used
- order_items
- products

## Skills shown
- Weekly time-based aggregation
- JOINs across product and transaction data
- Demand variability analysis
- CASE WHEN product classification
- Aggregations and filtering
- Business-focused inventory insights

## Questions answered
- Which products sell fastest versus slowest based on weekly demand?
- Which product categories have the highest return rates?
- Which products show consistent weekly demand?
- Which products have highly variable sales patterns?
- How can products be classified as fast, medium, or slow movers?
- Which products generate high revenue despite low unit volume?
