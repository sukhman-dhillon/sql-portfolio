# Project 01: Ecommerce Sales Performance

## Goal
Analyze ecommerce sales to understand revenue trends, AOV, and order value segments.

## Dataset
BigQuery public dataset: bigquery-public-data.thelook_ecommerce

## Tables used
- order_items
- products
- users

## Skills shown
- Aggregations (SUM, AVG, COUNT)
- Date grouping (DATE_TRUNC)
- JOINs
- CASE WHEN segmentation
- Filtering and sorting

## Questions answered
- What is monthly revenue and order volume?
- What is average order value (AOV) by month?
- Which products and categories drive revenue?
- How are orders distributed by value segment?
